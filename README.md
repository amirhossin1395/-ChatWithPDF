# 📄💬 چت با پی دی اف (Chat with PDF)

**ساخته شده توسط: امیرحسین حاجی‌مرادخانی | کرمان، ایران 🇮🇷**

## ✨ امکانات
- 📄 پشتیبانی از **PDF، Word، Excel و Text**
- 💬 **چت هوشمند** با محتوای فایل
- 🤖 **هوش مصنوعی** (DeepSeek) برای پاسخ‌دهی
- 🌍 پشتیبانی از **همه زبان‌ها**
- 🎨 رابط گرافیکی **Google Style**
- 📖 تشخیص خودکار **فصل‌ها**
- ⚡ اجرای **آفلاین** (پردازش روی دستگاه)

## 📥 نصب و اجرا
\```bash
pip install PyPDF2 python-docx openpyxl requests
python ChatWithPDF.py
\```

## 🧠 نحوه استفاده
۱. فایل PDF، Word، Excel یا Text رو باز کن
۲. بپرس: "فصل ۳ درباره چیه؟"
۳. بپرس: "خلاصه کتاب رو بگو"
۴. بپرس: "کلمات کلیدی چیه؟"

## 🏷️ تکنولوژی‌ها
Python | PyPDF2 | python-docx | openpyxl | DeepSeek API | Tkinter
